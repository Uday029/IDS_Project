# Real-Time Power BI Dashboard Guide

This guide will show you how to connect Power BI **directly to your live prediction database**. Whenever you use the Streamlit app's "Prediction Engine" to scan a new network packet, your Power BI dashboard will automatically reflect the new data!

We will use Power BI's native **Python Script Connector** to read directly from your SQLite database without needing to install any complex ODBC drivers.

## Step 1: Connect Power BI to the Live Database
1. Open Power BI Desktop.
2. Click on **Get Data** > **More...**
3. Search for **Python** and select **Python script**, then click Connect.
4. In the script box, copy and paste the exact code below. 
   *(Note: Make sure to replace `<YOUR_USERNAME>` with your actual Mac username if you are running Power BI on a Windows VM mapped to your Mac drive, or provide the exact path to the `ids_database.db` file on your system).*

```python
import sqlite3
import pandas as pd

# Path to the SQLite database generated by your Streamlit app
db_path = r'/Users/mac/.gemini/antigravity/scratch/IDS_Project/sql/ids_database.db'

# Connect to the database
conn = sqlite3.connect(db_path)

# Fetch all live predictions
query = "SELECT * FROM PredictionResults"
PredictionData = pd.read_sql(query, conn)

conn.close()
```
5. Click **OK**.
6. In the Navigator window, check the box next to **`PredictionData`** and click **Load**.

## Step 2: Build the Live Dashboard

Now that the live data is loaded, you can build visualizations that track your real-world predictions!

### 1. Total Live Scans (Card)
- **Visual**: Card
- **Field**: `id` (Set aggregation to **Count**)
- *Purpose*: Shows exactly how many packets you have scanned using the engine.

### 2. Live Threat Detection Rate (Donut Chart)
- **Visual**: Donut Chart
- **Legend**: `predicted_class`
- **Values**: `id` (Count)
- *Purpose*: Shows the real-time split between Secure (Normal) traffic and Threats.

### 3. Timeline of Scans (Line Chart)
- **Visual**: Line Chart
- **X-Axis**: `prediction_time`
- **Y-Axis**: `id` (Count)
- *Purpose*: Tracks when you are making predictions in the app over time.

### 4. Traffic Payload Distribution (Scatter Plot)
- **Visual**: Scatter Chart
- **X-Axis**: `src_bytes`
- **Y-Axis**: `dst_bytes`
- **Legend**: `predicted_class`
- *Purpose*: Plots the payload size of every live packet you scan.

### 5. Detailed Threat Log (Table)
- **Visual**: Table
- **Columns**: `prediction_time`, `username`, `protocol_type`, `service`, `predicted_class`, `model_used`
- *Purpose*: A live feed of all activity.

## Step 3: How to Auto-Update (Real-Time Refresh)

Whenever you use your Streamlit web app to make a new prediction:
1. Go to your Power BI Dashboard.
2. Click the **Refresh** button in the Home ribbon at the top.
3. Power BI will instantly run the Python script, fetch the newly added rows from the SQL database, and update all your charts immediately!

Save your dashboard as `Live_IDS_Dashboard.pbix`!
