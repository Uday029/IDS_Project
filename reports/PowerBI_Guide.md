# Power BI Dashboard Creation Guide

Since AI cannot natively generate `.pbix` binary files, follow these steps to build the professional Power BI dashboard for the IDS project.

## Step 1: Import Data
1. Open Power BI Desktop.
2. Click on **Get Data** > **Excel Workboo
3. Select `IDS_Project/excel/Cleaned_NSL_KDD.xlsx` (generated by the `data_cleaning.py` script).
4. Check the `Sheet1` box and click **Load**.

## Step 2: Create Measures (DAX)
Go to the **Modeling** tab > **New Measure** and create the following:

1. **Total Records**: 
   `Total Records = COUNTROWS('Cleaned_NSL_KDD')`
2. **Total Attacks**: 
   `Total Attacks = CALCULATE(COUNTROWS('Cleaned_NSL_KDD'), 'Cleaned_NSL_KDD'[attack_class] <> "Normal")`
3. **Total Normal Traffic**: 
   `Total Normal = CALCULATE(COUNTROWS('Cleaned_NSL_KDD'), 'Cleaned_NSL_KDD'[attack_class] = "Normal")`

## Step 3: Design the Dashboard Layout
Add a dark background or a professional template to match a cybersecurity theme. Use the **Format** pane for the page background.

### 1. KPI Cards (Top Row)
Add three **Card** visuals and place them at the top.
- Drag `Total Records` into Card 1.
- Drag `Total Attacks` into Card 2 (Format red).
- Drag `Total Normal` into Card 3 (Format green).

### 2. Donut Chart: Traffic Distribution
- **Visual**: Donut Chart
- **Legend**: `attack_class`
- **Values**: `Total Records`
- *Purpose*: Shows the percentage split between Normal, DoS, Probe, R2L, and U2R.

### 3. Bar Chart: Top 10 Attack Types
- **Visual**: Clustered Bar Chart
- **Y-Axis**: `label` (original attack name like neptune, smurf)
- **X-Axis**: `Total Records`
- Filter the visual in the Filter pane to show Top N = 10 by Total Records.

### 4. Column Chart: Protocol-wise Traffic
- **Visual**: Clustered Column Chart
- **X-Axis**: `protocol_type`
- **Y-Axis**: `Total Records`
- **Legend**: `attack_class` (to see the stack of attacks per protocol)

### 5. Table/Matrix: Service Vulnerability
- **Visual**: Matrix
- **Rows**: `service`
- **Values**: `Total Attacks`
- Sort by Total Attacks descending.

## Step 4: Add Slicers (Filters)
Add **Slicer** visuals on the left or top to allow interactivity:
1. Slicer 1: `protocol_type`
2. Slicer 2: `service`
3. Slicer 3: `attack_class`

## Step 5: Formatting and Polish
- Turn off backgrounds for visuals to blend with the dashboard background.
- Add descriptive titles to all charts.
- Use a coherent color scheme (e.g., Red for attacks, Green for normal, Blues for categories).

Save your file as `IDS_Dashboard.pbix` in the `dashboard/` folder!
